Usage Guidelines

✅

Do

Use on white, light gray, or dark navy backgrounds. Maintain clear space equal to the icon height on all sides.

❌

Don't

Don't stretch, rotate, recolor, or place on busy/patterned backgrounds. Don't use low-contrast color combinations.

📐

Minimum Size

Full logo: 160px wide minimum. Icon only: 24px minimum. Below these sizes use text only.

🔤

Typography

Wordmark uses Inter 700. Tagline uses Inter 400. Do not substitute with other fonts.

🌐

Web Use

Use SVG format for all web contexts. Use logo-compact for navbar (height 32–40px). Use logo-icon for favicon.

🎨

Icon Meaning

Hexagon = structure/system. 2×2 boxes = inventory. Yellow dot = AI intelligence. Arc + circles = shopping cart.
