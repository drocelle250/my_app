## 🚀 Next Steps

### For Local Development

1. **Backend:**
   ```bash
   cd Backend
   npm install
   npm run seed
   npm run dev
   ```
   Runs on: http://localhost:5000

2. **Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
   Runs on: http://localhost:5173
